Abstract

This project requires us to work with the Red Cross organisation, in order to extract structured data from documents. More specifically, extracting country information(admin 0,1...) from Disaster Response Emergency Fund(DREF) application documents. This entails the use of Natural Language Processing(NLP) techniques, therefore our initial research was centered around looking into already existing NLP libraries, compiling findings into a literature review.
