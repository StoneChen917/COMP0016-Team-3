Research

Our project's goal is to extract admin 0, 1 and 2, and historical data to place in a database in a table.

Text Summarization

Allows summarization of large paragraphs into shorter consumption without losing vital information. With text summarization, extracting essential information such as Country or the regionâ€™s name (Admin 0, 1, 2) will be much easier.
There are Extractive Text Summarization and Abstractive Text Summarization. The Extractive method uses exact sentences from the source to summarize the text, while abstractive generate phrases to summarize the text. To ensure that the data filled in the form is correct, Extractive Text Summarization seems like a better approach.
We researched several tools that we can use including AssemblyAI, plnia, Microsoft Azure, Meaning Cloud, gensim and sumy.

Named Entity Recognition

It involves the identification of key information in the text and classification into a set of predefined categories. An entity is basically the thing that is consistently talked about or refer to in the text.
Some of the categories that are the most important architecture in NER are person, organization and location. Can be used to identify details regarding provinces affected/targeted, funding in CHF etc.
