Project Background

IFRC helps National Societies apply for funds by filling out the DREF application. The existing GO platform helps with automating data entry for country specific information. However, it doesnâ€™t help with city/locality specific information. The project's goal is to extract admin 1/admin 2 into a table, so the employees can complete the form without unnecessary manual labour.

Client Requirements

The project has two parts: machine learning (the backend) and software engineering (the frontend). One key requirement for machine learning part is to use NLP to accurately find the name of the region (Admin 1, 2) The result shall be displayed to the user in a table in a database before the user decides whether to put the data in the go form or not.

Project Goals

This project requires us to build a program to carry out the application of retrieving region name from a word document which is the documentation for a disaster. The work entails initial research into existing methods of robust name entity recognition and solutions proposed for similar problems, preparing a research report from the findings, identifying the requirements, and proposing a solution application that can carry out name retrieval, and finally, development and deployment of the proposed system with necessary integrations.
