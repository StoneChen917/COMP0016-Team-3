Project Background

IFRC helps National Societies apply for funds by filling out the DREF application. The existing GO platform helps with automating data entry for country specific information. However, it doesnâ€™t help with city/locality specific information. The project's goal is to extract admin 1/admin 2 into a table, so the employees can complete the form without unnecessary manual labour.
Our task is to develop an import functionality which would allow national societies to upload PDF Documents, corresponding to disasters that have occurred, using an upload button. This is then pushed into fields within a database, structured by unique ISO codes. This will help keep better track of the disasters, establish historical trends, and allow National Societies to obtain grants for the implementation of Emergency responses at different admin levels within their territories.

Project Goals

This project requires us to build a program to carry out the application of retrieving region name from a PDF document which is the documentation for a disaster. The work entails initial research into existing methods of robust name entity recognition and solutions proposed for similar problems, preparing a research report from the findings, identifying the requirements, and proposing a solution application that can carry out name retrieval, and finally, development and deployment of the proposed system with necessary integrations.
